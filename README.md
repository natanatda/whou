# 프로젝트 소개

## 프로젝트 명
whoU

## 프로젝트 개요
개인화 직업 추천 서비스.
###
  - 진로에 대한 고민을 조언해줄 개인화 직업 추천 시스템의 필요성을 느낌.
  - 개인의 능력을 고려한 추천이 필요.
  - 기대 효과
    ###
    1. 빠른 진로 탐색
    2. 정년 퇴임 이후의 진로 모색
    3. 다양한 직업 정보 획득
    4. 진로 탐색 비용 절감
    5. 핵심 노동인구 고용 증가
<br>
